## Web
: WWW(world wide web)의 줄임말, 네트워크에 연결되어 정보 공유 가능한 서비스, 인터넷(국제 정보통신망)에서 작동

## 다양한 인터넷 서비스
- Protocol(Http, FTP, SMTP, POP, DHCP) 
- IP : 컴퓨터 식별주소 IPv4 IPv6
- DNS : IP주소를 맵핑한 문자열
- Port : 해당 컴퓨터에서 구동중인 프로그램 구분 번호
기본 포트번호 80

## 웹표준
: 웹 접근성을 높여줌
1. HTML : 웹 문서 기술 언어, 구조 및 정보 기술
2. CSS : 화면상 레이아웃 컨트롤 
3. Java Script : 동작 기술

## HTML
: 태그, 이름 속성 속성값을 가짐
1. 글자에 관련된 태그 : H1(제목), p(본문), br(개행), a(앵커), b, I, sup, del 태그
2. 목록 태그 : ul(순서없음), ol(순서있음), li
3. 표 태그 : table
4. 이미지 태그 : img
src : 이미지파일의 경로 지정
alt : 이미지가 없을 때 나오는 글자 지정
width & height
5. 오디오 태그 : audio
src : 오디오파일의 경로 지정
preload : 재생 전에 로드완료
auto play : 자동재생
loop : 반복
controls : 재생도구 출력
