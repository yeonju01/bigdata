## 인공지능
: 머신러닝, 딥러닝을 포함하는 개념으로 사고,학습 등 인간이 가진 지적능력을 컴퓨터로 구현하는 기술
tensorflow : 구글에서 딥러닝을 위해 제공하는 프레임워크
keras : 딥러닝 라이브러리

## 기계학습(머신러닝)
: 컴퓨터가 스스로 학습을 통해 학습모델의 여러 가중치를 찾아내도록 함(인풋과 아웃풋을 통해 컴퓨터가 프로그램을 작성)

1. 지도학습 
: 목적 값(Target Value)이 있음. 컴퓨터에게 답을 지정해주고 무엇이 맞는지 판단하도록(일정조건을 만들어 판단하도록) 데이터들을 활용하여 학습

- 분류(Classification) 
학습한 데이터를 기초로 컴퓨터가 결과를 판단하는 방법. 목적 값에 연속성이 없음(몇가지 값으로 분류됨)
(ex) 스팸 메일 고르기 -> 스팸이다/스팸이 아니다
- 예측(Regression)
일종의 회귀분석으로 F(x)=x+2와 같이 단순한 선형 회귀분석으로 예측 가능. 목적값에 연속성이 있음
(ex) 아파트 크기에따른 매매가격 분석 -> 조건(아파트 크기)에따라 결과(가격)를 보여줌

KNN : 최근접 이웃 알고리즘, 정확도가 높고 오류데이터가 결과값에 큰 영향을 끼치지 않으며 실 데이터 기반으로 비교하지만 느리고 메모리를 많이 사용함

- 딥러닝 : 레이어가 많음
옵티마이저(Optimizer) : Gradient Decent, Adam, RMSProp, ...
활성화 함수(activation function) : ReLU, Sigmoid, Softmax, ...
노드(node=인공뉴런) : 가중치가 있는 입력신호를 받음, 입력신호를 모두 더한 후 활성화 함수를 적용, 특정 값 이상인 경우에만 다음 노드의 입력값으로 전달
Batch 학습
DropOut
LearningRate

tensorflow2.0 - keras
(1)데이터셋 생성 : 훈련셋, 검증셋, 시험셋
(2)모델 구성(레이어 추가)
(3)모델 학습 과정 설정 : x(입력데이터), y(라벨값), batch_size(가중치 갱신할 샘플 갯수), epochs(학습 반복 횟수)
(4)모델학습
(5)학습과정 살펴보기(손실, 정확도 등 학습 상황 파악) : Model.fit() 히스토리기능
(6)모델 평가하기
(7)모델 사용하기
