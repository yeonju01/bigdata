## 웹 서비스
웹 서버 : 웹 브라우저로부터 URL을 받아 해당하는 HTML문서를 웹 브라우저로 보내주는 역할
- 애플리케이션 서버, 프로그램을 호출하여 처리 후 전송
- 아파치
애플리케이션 서버 : 기능을 실행 후 웹 서버에 전달
- 톰캣, 웹로직, Jboss
통합개발환경 : 에디터(코딩,실행)와 컴파일러를 연결하는 s/w
- Eclipse(J2EE), EditPlus

## 웹 개발에 필요한 학습
­JAVA : JAVA웹어플리케이션을 구현하기 위한 선행 학습 필요
­HTML : 웹 페이지를 구현하기 위한 기본 언어
­JavaScript : 클라이언트 기능을 구현하기 위한 언어
­Jquery : JavaScript의 대표적인 라이브러리로써, 클라이언트 사이드 스크립트 언어를 단순화 할 수 있다.
­CSS : 웹어플리케이션의 레이아웃 및 스타일을 지정하는 언어
